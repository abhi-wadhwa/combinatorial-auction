# Combinatorial Auction

Combinatorial auction engine with winner determination (ILP), VCG payments, and core pricing.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/combinatorial-auction.git
cd combinatorial-auction
make install
make run
make test
```

See [docs/](../docs/combinatorial-auction.tex) for the math.
